# Personalized-Fitness-coach-java


Simple desktop app demonstrating user login, user profile (height, weight, goals), personalized workout plan and progress tracking.


## Files
- DBConnection.java
- LoginUI.java
- WelcomeScreen.java
- Main.java
- User.java
- UserDAO.java
- Dashboard.java
- FitnessPlan.java
- README.md
- LICENSE


## How to run
1. Configure `DBConnection` with your DB credentials (MySQL / SQLite).
2. Create the database table with the SQL in `UserDAO` comments.
3. Compile: `javac *.java`
4. Run: `java Main`
